 # Симулятор распределенной системы

### Программа запускается через python main.py

Пример:
```
Добро пожаловать в симулятор распределенной системы.
Введите количество серверов:
3
Сервер 1: пусто
Сервер 2: пусто
Сервер 3: пусто
Команда: добавить 10
Задание с 10 секундами выполнения направлено на Сервер 1.
Команда: добавить 10Состояние серверов
Задание с 10 секундами выполнения направлено на Сервер 2.
Команда: добавить 10
Задание с 10 секундами выполнения направлено на Сервер 3.
Команда: добавить 10
Задание с 10 секундами выполнения добавлено в очередь. [10]
Команда: добавить 10
Задание с 10 секундами выполнения добавлено в очередь. [10, 10]
Команда: добавить 10
Задание с 10 секундами выполнения добавлено в очередь. [10, 10, 10]
Команда: добавить 10
Задание с 10 секундами выполнения добавлено в очередь. [10, 10, 10, 10]
```
